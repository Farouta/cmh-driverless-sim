# Getting started with MkDocs

Tutorial to help get users started with mkDocs

## Installation

To install MKDocs with pip, run the following command:

```console
pip install mkdocs
```

For other ways of installations, follow this link : https://www.mkdocs.org/getting-started/ 

## Getting Started

Please put all the markdown files required for documentation into the already provided docs folder. To run the site, first run the command 

```console
mkdocs build
``` 
in your terminal in the directory with `mkdocs.yml`. After this has run sucessfully, run in the same directory:


```console
mkdocs serve
```

This will 
run a site locally. Click on the link it gives, now you will be able to see the site! You can add changes to your site while it is  is running, it will update automatically.
